5. Material Symbols style family selection: formality <= 4 -> Rounded, formality >= 7 -> Sharp, else -> Outlined.
6. Output: `{ library, default_weight, emphasis_weight, style_family (Material Symbols only), cdn_url, usage_syntax }`

Include explicit boundary values (use >= and <= with inclusive bounds) so the architect agent faces zero ambiguity. Reference the differentiation seed axes from `core/references/design-inputs.md`.

**Section 4: Icon Color** -- Brief section documenting that icons use `currentColor` inheritance. No `--icon-color-*` tokens needed. Document: "Apply color via `color: var(--text-secondary)` on the parent or icon element. Icons inherit via CSS `currentColor`."
