   - Pairing A "Editorial Commerce": Syne 700 (display, -0.02em) + Work Sans 400/500 (body, 0em) -- distinctive personality + excellent readability. Both on Google Fonts.

   - Recommended: Spacious for product browsing, compact for cart/checkout (dual density)
   - Concrete Values table: Product grid gap 16-24px, card internal padding 16px, cart item row height 72-80px, section gap 32-48px, form field gap 16px, button padding 12px 24px, touch target 48x48px

   - **CartItem** (name MUST be exactly "CartItem"): Product thumbnail, title, variant info (size/color chips), quantity selector (+/-), line total, remove button. Include structure, dimensions (compact for sidebar cart overlay), states (default/updating-quantity/removing with animation/error), tap-target
   - **PriceDisplay** (name MUST be exactly "PriceDisplay"): Price with currency symbol, optional compare-at price (struck through), savings badge (percentage or amount), installment info line. Include structure, dimensions, states (regular/on-sale/sold-out/loading), formatting rules

9. **Accessibility Specifics:** E-commerce-specific -- product images require descriptive alt text (not just "product image"), price announced with currency by screen reader, cart count announced as live region on update, color/size selectors announce selected state, checkout progress communicated via aria-current
10. **Border Radius:** Token table (sm=4px, md=8px, lg=12-16px) -- moderate to friendly, approachable without being childish. Product images may use slightly more rounding.
11. **Shadow Style:** Layered, multi-depth shadows matching generate-system.md "layered" designation. Product card hover elevation, modal depth, dropdown overlays. Creates the "physical product" feel. Token table with full CSS values (multi-layer shadow syntax).
12. **E-commerce-Specific Additions:** Product image gallery patterns (thumbnail strip, zoom, lightbox), size/variant selectors (chip groups with out-of-stock states), cart count badges (animated on update), sale price formatting (original vs sale with savings percentage), star rating display (half-star precision, count display), free shipping progress bar, trust badges (payment icons, security seals), breadcrumb category navigation
