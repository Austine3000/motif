## Cross-Phase Integration Results

### Critical Issues

#### 1. Vertical Path Mismatch (Severity: Critical)

**All 4 vertical reference files are installed but never consulted by workflows/agents.**

- **Installed to:** `.claude/get-motif/references/verticals/{vertical}.md`
- **Workflows look for:** `.claude/get-motif/verticals/{vertical}.md` (missing `references/` segment)
- **Agents hardcode:** `.claude/get-motif/verticals/{vertical}.md` (same wrong path)
- **Effect:** The research workflow checks `if VERTICAL_REF EXISTS` — it won't exist at the expected path, so vertical intelligence is silently skipped. This breaks Motif's core value proposition.
- **Affected files:**
  - `core/workflows/research.md` (lines 36, 57, 89)
  - `core/workflows/generate-system.md` (lines 35, 54)
  - `runtimes/claude-code/agents/motif-researcher.md` (line 50)
  - `runtimes/claude-code/agents/motif-system-architect.md` (line 52)
- **Fix:** Either update installer copy mapping to install verticals at `get-motif/verticals/` (flat), or update all workflow/agent paths to include `references/`.

#### 2. CI Workflow Non-Existent Action Versions (Severity: High)

- `.github/workflows/publish.yml` uses `actions/checkout@v6` and `actions/setup-node@v6`
- Neither v6 exists — latest is v4
- **Effect:** Publish flow will fail immediately on GitHub Actions
- **Fix:** Change both to `@v4`

### Medium Issues

#### 3. init.md References Missing CLAUDE-MD-SNIPPET.md (Severity: Medium)

- `runtimes/claude-code/commands/motif/init.md` line 212 references `templates/CLAUDE-MD-SNIPPET.md`
- This file exists at `runtimes/claude-code/CLAUDE-MD-SNIPPET.md` but is not in the installer copy mapping
- The installer handles CLAUDE.md injection itself, so this is partially mitigated for npx users
- **Fix:** Either remove the init.md reference (installer already handles it) or add the file to the copy mapping

### Low Issues

#### 4. Stale Package Name References (Severity: Low)

- `bin/install.js` help text (lines 31-44): says `npx motif@latest` instead of `npx motif-design@latest`
- `core/references/runtime-adapters.md` (lines 90-94): same stale reference
- **Fix:** Update both to `npx motif-design@latest`
